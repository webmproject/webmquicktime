WebM QuickTime v0.3.0.

This release bundles the XiphQT component with the WebM QuickTime installer to provide support for encoding and decoding the Vorbis audio streams contained within WebM files. Other changes include:

- The components will now autoupdate.
- Improvements to progressive download playback performance.
- Fix incorrect reporting of file duration during initial track load.
- Fix some timestamp rounding issues that resulted in inaccuracy.
- Support lacing.
